You will adjust the width of this table data cell so that it aligns nicely with the image at the left. You need to know that the total width of this table is 900 pixels. The table uses 5 pixels of cellpadding and is two data cells wide so there are 20 pixels of cellpadding to account for.

You will take 900 pixels minus 20 pixels of cellpadding minus xxx pixels (photo width = width of cell for description text. The width setting is an attribute within the table data cell td code. It is coded as Style="width:xxxpx". You will change the xxx to the number of pixels wide that you would like this table data cell to be.
